🎙️ Captions (.srt)

              

                Generate a YouTube-uploadable .srt from the
                video's audio using local Whisper. The DaVinci timeline is
                not modified — you upload the file in
                YouTube Studio → Subtitles.
              

              

                
                
                
              

            


            

              
                ⚠️ Settings that the YouTube Data API does NOT expose (must be set in YouTube Studio after upload)
              
              

                These fields exist only in YouTube Studio — there is no public
                API to set them. After publishing, open the video in Studio
                and configure:
                
    
                  
  • Shorts sampling (Allow / Don't allow)
    • 
                  
  • Comments — Allow all / Hold for review / Disable
    • 
                  
  • Comment ranking — Top comments / Newest
    • 
                  
  • Automatic chapters — On / Off
    • 
                  
  • Caption certification
    • 
                  
  • Self-rated content categories (Inappropriate language, Adult content, Violence, Harmful or dangerous acts, Hateful & derogatory content, Recreational drugs, Firearms, Controversial issues, Tobacco)
    • 
                

                Title / description / tags / category / thumbnail / playlists /
                privacy / made-for-kids / synthetic media / language /
                recording date / embedding ARE handled here automatically.
              

            

            

              
                🃏 Cards (also Studio-only)
              
              

                The YouTube Data API has no endpoint for in-video cards
                (link to playlist / video / channel). You must add cards
                manually in YouTube Studio → Editor → Cards. Playlists you
                might want to link can be loaded via the Playlists section
                above (their IDs / titles will be in the publish result for
                copy-paste).
